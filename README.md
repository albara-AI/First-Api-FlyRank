# Book Library API

A minimal FastAPI backend with JSON endpoints — built as a first hands-on task in the FlyRank Backend AI Engineering program.

## Tech Stack

- Python 3
- FastAPI
- Uvicorn (ASGI server)

## Setup & Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Start the server
uvicorn main:app --reload
```

The server runs at `http://127.0.0.1:8000`

## Endpoints

| Method | Path               | Description                          |
|--------|--------------------|--------------------------------------|
| GET    | `/`                | Welcome message                      |
| GET    | `/books`           | List all books with count            |
| GET    | `/books/{book_id}` | Get a book by ID (validated as int)  |
| GET    | `/search?q=...`    | Search books (defaults to "All")     |

## Try It

**From the browser:**

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/books
- http://127.0.0.1:8000/books/3
- http://127.0.0.1:8000/search?q=llm
- http://127.0.0.1:8000/docs — interactive Swagger UI (auto-generated by FastAPI)

**From curl:**

```bash
curl http://127.0.0.1:8000/
curl http://127.0.0.1:8000/books
curl http://127.0.0.1:8000/books/3
curl "http://127.0.0.1:8000/search?q=llm"
```

## Example Response

```json
{
  "book_id": 3,
  "status": "available"
```
